JSON.parse() and JSON.stringify() Demo

See the browser console to see the results:

const jsonString = `{"user": "Bill Gates"}`;

console.log(`${typeof jsonString} jsonString: ${jsonString}`);

const jsonObject = JSON.parse(jsonString);

console.log(`${typeof jsonObject} jsonObject: ${jsonObject.user}`);

console.log(`${typeof jsonObject} jsonObject: ${JSON.stringify(jsonObject)}`);
